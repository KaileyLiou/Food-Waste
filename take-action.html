Take Action

There are many ways you can get involved.

What You Can Do At Home

  • Store food properly
  • Eat leftovers
  • Keep track of expiration dates
  • Compost food scraps
  • Keep in mind portion sizes so that you don't end up wasting extra food

Start an Initiative to Spread the Word

  • Host an event or organize a workshop
  • Make a campaign
  • Start a program
  • Advocate for policy changes
  • Create a website
  • Build an app

My Food Donation App Design

How have I been getting involved?

I created a UI/UX design of an app called MeaningfulMeals.

This design was created using Figma and Canva. The goal of this app is to reduce food waste while providing healthy meals to those in need.

It is still an ongoing project that I will hopefully be able to complete soon!
